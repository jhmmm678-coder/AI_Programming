AI-Powered Financial News Sentiment Dashboard

A lightweight dashboard that analyzes financial news headlines/summaries using a pre-trained model (FinBERT) and assigns both sentiment (negative / neutral / positive) and simple issue tags. It helps retail investors quickly capture market mood and key issues without reading every article.

Model: ProsusAI/finbert App: Streamlit Input: CSV upload or manual paste Output: charts + table + CSV download

Issue Tags (Rule-based)

Regulation / Legal Risk

  • keywords: lawsuit, regulation, fine, etc.

Earnings

  • keywords: earnings, profit, loss, etc.

Growth

  • keywords: growth, market share, etc.

ESG / Climate

  • keywords: carbon, renewable, solar, etc.

If no keyword matches, the tag defaults to Other.

AI Model & Motivation

Model: ProsusAI/finbert

  • BERT-based transformer specialized for financial sentiment
  • Outputs class probabilities: negative / neutral / positive
  • Used as pre-trained weights (no training from scratch)

FinBERT is preferred over generic sentiment models because it is trained on financial domain text.

How it is used

  • Tokenize each news text
  • Forward pass through FinBERT
  • Get probabilities (neg/neu/pos)
  • Choose label with highest probability

System Pipeline

1
Input

Upload CSV (with text column) or paste texts (one per line).

2
Preprocess

Lowercase internally; truncate to max length (256 tokens).

3
FinBERT

Predict sentiment label + probability for each news item.

4
Issue Tagging

Keyword rules assign tags (Legal/Earnings/Growth/ESG/Other).

5
Aggregate

Count sentiments + tags; summarize overall mood and issues.

6
Visualize

Bar charts + detailed table; allow CSV download of results.

Input: CSV or pasted texts → preprocess (lowercase, truncate) → FinBERT sentiment (neg/neu/pos + probs) → rule-based issue tag → aggregation (counts) → Streamlit UI (charts + table + download)

How to Run Locally

Install

python -m venv .venv # Windows: .\.venv\Scripts\activate # macOS/Linux: # source .venv/bin/activate pip install -r requirements.txt

Launch Streamlit

streamlit run app.py

Input either a CSV file (must include a text column) or paste news texts manually.
